Transformando String em Number

Para realizar a conversão para Number. é necessario usar number.parseInt() fora do window.

Há três representações de Number.
1-Number.parseFloat(window.prompt(n)) -> decimal
2-Number.parseInt(window.prompt(n)) -> inteiro
3-Number(prompt(n)) -> a máquina escolhe

Usar ${} Deixa o código mais limpo do que a concatenação normal.
